### 项目原理

本项目通过在镜像站上传 `JS` 库文件的方式，在引导页再引入该 `JS` 文件。如果引入成功，则计算时间；如果引入失败，则代表无法访问。

#### 实现步骤：

1. 上传一个 `JS` 库文件到 `GF` 网站。
2. 在导航页面使用 `<script>` 标签引用该库。
3. 如果 `JS` 成功执行，计算从插入 `JS` 到执行 `JS` 的时间差，即为延迟。

不同的镜像站有不同的域名，代表有不同的库脚本引用地址，可以用于计算不同镜像站的延迟。
